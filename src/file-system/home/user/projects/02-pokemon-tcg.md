## 2025

### • Vite, TypeScript,

The Pokemon TCG website is a web-based game where you can collect Pokemon cards through a gacha system. This website is completely free—there are no fees or in-app purchases. I created this website because I didn't have the money to buy Pokemon TCG cards in the real life. I found the API source for this website on GitHub, but I've forgotten the link.
