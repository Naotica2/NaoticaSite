



## *NaoFlix*
## 2026

### • React Native, TypeScript,
### • postgreSQL, APK
Naoflix is an anime streaming app, but it offers more than just anime. The Naoflix app also provides movie streaming and comics that can be read for free. Of course, Naoflix is simply a learning project that I've published, and the data on Naoflix is managed by a third-party server, so Naoflix is merely a frontend for that data.
